Detailed Description

Processing thread for a single relay pipeline.

Reads from the relay's RawDataRingBuffer, parses SER responses, stamps each record with the relay's identity (relay_id + relay_name), inserts into the shared SQLite database, and broadcasts via the shared WebSocket server.

The key difference from the original ProcessingWorker is the relay_id / relay_name stamping on every parsed SERRecord before DB insertion.
