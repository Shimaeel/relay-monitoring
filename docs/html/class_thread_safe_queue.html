Detailed Description

template<typename T>
class ThreadSafeQueue< T >

Thread-safe queue for producer-consumer workflows.

Template Parameters
TType of elements stored in queue

Provides mutex-protected enqueue/dequeue operations with condition-variable signaling for efficient waiting.
