Detailed Description

Main entry point for Telnet-SML Substation Monitoring System.

This application provides automated monitoring of electrical substation equipment via Telnet protocol, using a Boost.SML finite state machine for robust connection management and System Event Record (SER) data collection.

System Architecture

Application Workflow

Key Features

  • Automatic connection and authentication to substation relays
  • Retry logic with configurable attempts and delays
  • SQLite persistence for System Event Records
  • Real-time WebSocket server for UI updates
  • Clean state machine architecture using Boost.SML
