Detailed Description

Telnet-SML application facade (public API).

Defines the public interface for the Telnet-SML application. The facade hides the multi-threaded implementation behind a small, stable API intended for simple start/stop orchestration.

Architecture Overview

Thread Model

The application spawns multiple threads on start():

Thread Purpose
Reception Telnet I/O with relay
Processing Parse + DB + broadcast
WebSocket Browser communication
Poller Periodic data fetch
SHM Reader JSON file output

Usage Example

#include "telnet_sml_app.hpp"
int main() {
TelnetSmlApp app;
if (!app.start()) {
return 1;
}
app.waitForExit();
app.stop();
return 0;
}
