Detailed Description

Complete processing pipeline for a single relay device.

Owns a TelnetClient, RawDataRingBuffer, ReceptionWorker, and ProcessingWorker. Holds references to the shared SERDatabase, SERWebSocketServer, and SharedRingBuffer.

Created on demand by RelayManager when the user activates a relay from the dashboard. Destroyed when the user deactivates the relay or the application shuts down.

Lifecycle

auto pipeline = std::make_unique<RelayPipeline>(config, db, ws, shm, running);
pipeline->start(); // Connects + starts polling
pipeline->queueCommand("SER");
// ...
pipeline->stop(); // Graceful shutdown
