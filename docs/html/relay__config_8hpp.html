Detailed Description

Relay device configuration registry.

Defines the static configuration for all known relay devices in the substation monitoring system. Each relay has a unique ID, network address, credentials, and display metadata.

Adding a New Relay

To add a new relay, append a RelayConfig entry to the vector returned by getRelayConfigs(). Each relay must have a unique id.
