Detailed Description

DATE synchronization manager for relay ↔ local-PC time operations.

Orchestrates reading relay time, comparing with local PC time, and synchronizing the relay clock. Designed to be called from the WebSocket message handler so that incoming JSON actions are processed without blocking the main I/O thread.

Supported WebSocket Actions

Action Description
read_time Read relay DATE + local PC time
sync_time Write local PC time to relay
