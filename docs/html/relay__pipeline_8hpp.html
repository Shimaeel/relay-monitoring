Detailed Description

Per-relay processing pipeline for on-demand relay connections.

Bundles all per-relay components (TelnetClient, workers, ring buffer) into a single RelayPipeline object. The pipeline is created on demand when the user clicks a relay card in the browser, and torn down when the relay is disconnected.
