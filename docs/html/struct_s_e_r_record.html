Detailed Description

Structure representing a System Event Record (SER).

Holds parsed data from a single SER entry retrieved from a substation relay device. Each record represents one event in the relay's event log.

Field Mapping

Field Source Example
record_id Event # "45"
timestamp Date + Time "02/14/22 12:47:19.970"
status State "Asserted" or ""
description Element "SALARM"
Note
record_id is stored as string to preserve original relay numbering
status may be empty for events without assertion state
