Detailed Description

Thread-safe wrapper around TelnetClient for DATE operations.

All public methods lock the internal mutex so that the ReceptionWorker's command queue and the TimeSyncManager can share the same underlying TelnetClient safely (only one caller at a time sends Telnet I/O).

Usage (inside ReceptionWorker or ProcessingWorker)

RelayService relay(client);
auto [ok, dt] = relay.readRelayTime();
if (ok)
std::cout << "Relay time: " << dt << "\n";
