Detailed Description

Thread-safe SPSC ring buffer for variable-length binary payloads.

Provides a circular buffer where each payload is prefixed with a 4-byte little-endian length header. Designed for efficient inter-thread communication with minimal synchronization overhead.

Thread Safety
  • Single producer thread calls write()
  • Single consumer thread calls waitRead()
  • Multiple threads may call notifyAll() for coordinated shutdown
Memory Model
Uses acquire-release semantics for lock-free position updates. Condition variable provides efficient blocking for the consumer.
Example Usage
// Initialize buffer (500KB capacity)
SharedRingBuffer ring(500 * 1024);
// Producer thread
void producer() {
std::vector<uint8_t> payload = encodeData();
if (!ring.write(payload.data(), payload.size())) {
// Handle write failure (payload too large)
}
}
// Consumer thread
void consumer(std::atomic<bool>& stop) {
std::vector<uint8_t> data;
while (!stop.load()) {
if (ring.waitRead(data, stop)) {
processPayload(data);
}
}
}
// Shutdown
void shutdown() {
stop.store(true);
ring.notifyAll(); // Wake consumer
}

bool SharedRingBuffer::waitRead ( std::vector< uint8_t > & out,
std::atomic< bool > & stop_flag )
inline

Waits for and reads a payload from the buffer (blocking).

Blocks the calling thread until one of:

  • A payload becomes available (returns with data)
  • The stop_flag is set externally (returns false)
Parameters
[out]outVector to receive the payload data. Cleared and resized automatically.
[in]stop_flagAtomic flag checked during wait. Set to true externally to abort the wait.
Return values
trueSuccessfully read a payload into out.
falseStopped (stop_flag set) or internal error.
Precondition
Consumer thread owns this call exclusively.
Postcondition
If true: out contains the payload; read position advanced.
If false: out may be in an undefined state.
Thread Safety
Must be called from a single consumer thread only.
Blocking Behavior
Uses condition_variable::wait() internally. Call notifyAll() from another thread to unblock.

bool SharedRingBuffer::write ( const uint8_t * data,
std::size_t len )
inline

Writes a payload to the buffer (non-blocking).

Appends a length-prefixed payload to the ring buffer. If insufficient space exists, oldest payloads are automatically dropped until enough room is available.

Parameters
[in]dataPointer to payload data (must not be nullptr).
[in]lenLength of payload in bytes (must be > 0).
Return values
trueWrite succeeded.
falseWrite failed due to:
  • nullptr data pointer
  • Zero length
  • Payload exceeds maximum size (capacity - 4)
Postcondition
If true: Data is available for consumer.
If data was dropped: overwriteOccurred() returns true.
Note
This method NEVER blocks the calling thread.
Maximum payload size: capacity() - kHeaderSize bytes.
Thread Safety
Must be called from a single producer thread only.
