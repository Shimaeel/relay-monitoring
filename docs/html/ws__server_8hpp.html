Detailed Description

WebSocket Server for Real-time SER Data Access.

This header implements a WebSocket server using Boost.Beast for providing real-time access to System Event Records from the web UI.

Usage Example

SERDatabase db("records.db");
db.open();
SERWebSocketServer wsServer(db, 8765);
wsServer.start();
// Server now listening on ws://localhost:8765
// ... application runs ...
wsServer.stop();
