Detailed Description

Example Application Using Telnet-SML DLL.

Demonstrates how to use the telnet_sml.dll library in an external application. This example connects to a relay, retrieves SER data, and displays it through the WebSocket server.

Building This Example

# First build the DLL
./build_dll.ps1
# Then compile this example
g++ -std=c++17 ^
-I third_party/sml/include ^
-I C:\Libraries\boost_1_90_0 ^
-I C:\sqlite ^
dll_example.cpp ^
-Llib -ltelnet_sml ^
-lws2_32 -lmswsock ^
-o dll_example.exe
# Copy DLL to executable location
copy lib\telnet_sml.dll .
# Run
./dll_example.exe
