std::vector< uint8_t > asn_tlv::encodeSerRecordsToTlv ( const std::vector< SERRecord > & records)
inline

Encode SER records to ASN.1 BER/TLV format.

Creates a top-level APPLICATION 1 TLV containing zero or more SEQUENCE TLVs, each with context-specific fields for record data.

Output Structure

0x61 [len] {
0x30 [len] {
0x80 [len] record_id
0x81 [len] timestamp
0x82 [len] status
0x83 [len] description
}
... (more records)
}
Parameters
recordsVector of SERRecord to encode
Returns
std::vector<uint8_t> Binary TLV payload
