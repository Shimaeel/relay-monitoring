Detailed Description

On-demand coordinator for multiple relay pipelines.

RelayManager holds the static relay configuration registry and creates / destroys RelayPipeline instances on demand. The browser UI sends a WebSocket message to start or stop a specific relay, and this class translates that into pipeline lifecycle operations.
