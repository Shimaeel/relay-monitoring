Detailed Description

Password management for relay LEVEL1 / LEVEL2 password changes.

Provides validation, ASN.1 BER encoding (Tag 34), and relay communication for password change operations. Designed to be called from the WebSocket action handler so that incoming JSON requests are processed without blocking the main I/O thread.

Supported WebSocket Actions

Action Description
change_password Change relay password (LEVEL1 / LEVEL2)
