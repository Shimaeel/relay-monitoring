Detailed Description

Validates and executes relay password changes (LEVEL1 / LEVEL2).

Provides a high-level interface for:

  • Validating password values against the allowed set (TAIL1 / OTTER1)
  • Building ASN.1 Tag 34 encoded payloads
  • Sending password change commands to the relay
  • Handling WebSocket JSON requests end-to-end

Thread-safety: each public method acquires RelayService's internal mutex through the RelayService calls, so this class itself is safe to call from any thread (e.g. posted on a Boost.Asio strand).

WebSocket Protocol

Request:

{
"action": "change_password",
"level": "LEVEL1",
"value": "TAIL1"
}

Success Response:

{ "action": "change_password", "status": "success" }

Failure Response:

{ "action": "change_password", "status": "failed", "error": "..." }
